AtmaCinema adalah aplikasi mobile berbasis Flutter yang menyediakan informasi tentang film di bioskop, fitur pembelian tiket, dan pengalaman personalisasi untuk pengguna. Aplikasi ini menggunakan backend API untuk mengelola data secara real-time.

#### Fitur Utama
- Film: Menampilkan Now Playing dan Coming Soon.
- Menu : Menampilkan informasti tentang makanan dan minuman yang tersedia di bioskop.
- Studio : Menampilkan informasi mengenai studio yang tersedia di bioskop.
- Profil Pengguna : Menampilkan informasi terkait data pengguna dan menyediakan fitur edit profile (menggunakan kamera untuk edit foto profile) dan fitur logout.
- Wishlist : Menyimpan daftar film yang diinginkan.
- Detail Film : Informasi mendalam tentang film tertentu, termasuk sinopsis, rating, genre, dan lainnya.
- Tiket : Menyediakan layanan pembelian tiket, pembayaran, dan cetak tiket dalam format PDF.
- Histori : Melihat riwayat pembelian tiket.
- Search Menu : Menyediakan layanan text-to-speech atau teks biasa untuk mencari menu yang tersedia di bioskop.
